Intrucciones para acceder al proyecto:
Para poder ejecurar el proyeto se debe ejecutar el archivo app.py en VS Code, y undir ctrl+click sobre la URL http://127.0.0.1:5000 
esto lo redireccionará a la pagina web.

¡¡Nota importante!!
para poder ejecutar el proyecto correctamente se debe tener varias librerias instaladas, para acortar, en la terminal de VS Code ejecute esta linea de codigo:
pip install Flask Flask-SQLAlchemy SQLAlchemy plotly pandas numpy pyobdc sys os
